# react-native-thermal-printer-driver

React Native thermal printer driver for ESC/POS printers over Bluetooth Classic, BLE, and TCP

## Installation


```sh
npm install react-native-thermal-printer-driver
```


## Usage


```js
import { multiply } from 'react-native-thermal-printer-driver';

// ...

const result = multiply(3, 7);
```
